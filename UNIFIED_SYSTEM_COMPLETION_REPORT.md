## 🔌 API 接口架构

### 统一认证 API

- `POST /api/auth/login` - 统一登录（支持角色选择）
- `GET /api/auth/me` - 获取用户信息（根据角色）
- `POST /api/auth/logout` - 退出登录

### 家长端 API

- `GET /api/parent/dashboard` - 家长仪表板数据
- `POST /api/parent/submit` - 提交改卷请求
- `GET /api/parent/results` - 获取成绩列表
- `GET /api/parent/result/:id` - 获取成绩详情
- `GET /api/parent/history` - 获取历史记录

### 教师端 API

- `GET /api/teacher/dashboard` - 教师仪表板数据
- `POST /api/teacher/tasks` - 创建自动化任务
- `GET /api/teacher/tasks` - 获取任务列表
- `GET /api/teacher/tasks/:id` - 获取任务详情
- `POST /api/teacher/tasks/:id/execute` - 执行任务
- `POST /api/teacher/tasks/:id/cancel` - 取消任务
- `GET /api/teacher/tasks/:id/status` - 获取任务状态
- `GET /api/teacher/tasks/:id/statistics` - 获取统计数据
- `GET /api/teacher/tasks/:id/analytics` - 获取分析数据
- `DELETE /api/teacher/tasks/:id` - 删除任务

### 系统 API

- `GET /health` - 系统健康检查
- `POST /api/upload` - 文件上传
