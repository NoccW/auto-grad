## 🔌 API 接口

### 教师端 API

- `POST /api/teacher/tasks` - 创建任务
- `GET /api/teacher/tasks` - 获取任务列表
- `GET /api/teacher/tasks/:id` - 获取任务详情
- `POST /api/teacher/tasks/:id/execute` - 执行任务
- `POST /api/teacher/tasks/:id/cancel` - 取消任务
- `GET /api/teacher/tasks/:id/status` - 获取任务状态
- `GET /api/teacher/tasks/:id/statistics` - 获取统计数据
- `GET /api/teacher/tasks/:id/analytics` - 获取分析数据
- `DELETE /api/teacher/tasks/:id` - 删除任务

### 系统 API

- `GET /health` - 健康检查
- `GET /api/auth/me` - 用户信息
- `POST /api/upload` - 文件上传

## 📝 使用说明

### 1. 启动系统

按照部署说明启动前后端服务

### 2. 访问界面

打开浏览器访问 http://localhost:5174

### 3. 创建任务

1. 进入教师主页
2. 点击"创建新任务"
3. 填写目标网站信息
4. 设置账号密码和试卷数量
5. 启动任务

### 4. 监控执行

1. 在任务列表中查看状态
2. 点击"监控"进入实时监控页面
3. 查看执行进度和日志

### 5. 分析结果

1. 任务完成后点击"分析"
2. 查看分数分布和错误分析
3. 导出分析报告
